Intercom Canvas App for Smart Image Replacement

How it works:

Analyzes articles using AI to find matching images
Visually compares images regardless of URL differences
Replaces all matching instances automatically
Provides detailed feedback on replacements made

Setup Instructions:

1. Environment Variables:

Set INTERCOM_ACCESS_TOKEN with your Intercom API token

2. Intercom App Configuration:

Configure this app as a Canvas app in your Intercom developer console

3. Required Permissions:

Read articles and Write articles

Usage: This app runs as a Canvas component within Intercom. Install it in your workspace and access it from the Intercom inbox.
